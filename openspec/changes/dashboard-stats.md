# Proposal: Dashboard / 统计

## 背景与目标

为管理员提供系统概览页面，展示 Mango BaaS 的关键指标和统计数据。

## 用户故事

1. 作为管理员，我希望一眼看到系统的总体状态
2. 我希望看到各项目的 API 调用量排行
3. 我希望看到最近的审计日志活动

## 功能设计

**统计面板：**
- 总项目数
- 总用户数
- 总 API 调用次数（今日/本周/本月）
- 活跃项目排行

**图表：**
- API 调用趋势图（7天/30天）
- 各项目调用量占比

**最近活动：**
- 最近创建的 API Keys
- 最近注册的 Webhooks
- 最近的操作日志

## 成功标准

1. Dashboard 页面展示关键指标
2. 数据自动刷新（可配置间隔）
3. 点击指标可跳转到详情页

## 不做会怎样

管理员无法直观了解系统运行状态。
